Blood Bank Management System
A sleek, browser-based management tool built for the Cognebula'26 challenge. It streamlines blood donor tracking and inventory management through an intuitive, dark-mode interface.

Core Features
Real-time Inventory: Live tracking of 8 blood groups with visual low-stock alerts.

Transaction Logic: Simulate blood donations and requests with instant stock updates.

Smart Validation: Built-in age eligibility checker for potential donors.

Zero-Backend Persistence: Data stays saved using localStorage for a seamless experience without a server.

Tech Used
Frontend: HTML5, CSS3 (Modern Grid/Flexbox)

Logic: Vanilla JavaScript (ES6+)

Deployment: GitHub Pages
